# swarm

### requirements

`Ruby 2+`

### description

Squish or be squished!

### quick installation

```
gem install the-swarm
```

### usage

`swarm` or `SWARM=easy swarm`

### building and installing from source

```
git clone https://github.com/glucero/swarm
cd swarm
bundle
gem build swarm.gemspec
gem install the-swarm-VERSION.gem
```
